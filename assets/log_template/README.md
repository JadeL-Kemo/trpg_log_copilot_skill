# TRPG Log Template

This directory contains templates for the TRPG log file structure.

Templates are generated automatically by `scripts/init_session.py` — prefer using the script over manual copying.

To manually use these templates:
1. Copy all files from this directory to your session directory
2. Rename/merge as needed
3. Fill in the placeholders

Key files:
- `00_当前局势.md` — Dashboard (loaded every turn)
- `01_线索板.md` — Active clues board
- `07_跑团规范手册.md` — Self-contained migration manual (most important for platform switching)
