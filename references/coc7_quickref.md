# CoC 七版关键规则速查

> 用于车卡和跑团过程中快速参考。非完整规则书，仅核心机制。

## 一、属性（Characteristics）

### 八项基础属性

| 属性 | 英文 | 范围 | 决定 |
|------|------|------|------|
| STR 力量 | Strength | 15-90 | 近战伤害加成、力量对抗 |
| DEX 敏捷 | Dexterity | 15-90 | 闪避基础值(DEX/2)、先攻、射击 |
| CON 体质 | Constitution | 15-90 | HP、毒抗、耐力 |
| APP 外貌 | Appearance | 15-90 | 社交第一印象、魅惑基础值(APP/2) |
| POW 意志 | Power | 15-90 | SAN 初始值(POW)、魔法值(MP=POW/5)、意志对抗 |
| SIZ 体型 | Size | 40-90 | HP(SIZ+CON)/10、伤害加成体型 |
| INT 智力 | Intelligence | 40-90 | 兴趣技能点数(INT×2)、灵感检定 |
| EDU 教育 | Education | 40-90 | 职业技能点数、知识检定 |

### 属性检定方式

- **常规检定**：d100 ≤ 属性值 = 成功
- **困难检定**：d100 ≤ 属性值/2 = 成功
- **极难检定**：d100 ≤ 属性值/5 = 成功

## 二、技能点数

### 职业技能点数
```
警察探员/侦探：EDU×2 + DEX×2（或 STR×2）
学者/教授：EDU×4
记者：EDU×2 + DEX×2（或 APP×2）
医生：EDU×4
```

### 兴趣技能点数
```
个人兴趣点数 = INT × 2
```

### 技能上限

| 类型 | 默认上限 | 说明 |
|------|----------|------|
| 职业技能 | 75% | 可经 KP 同意放宽至 80% |
| 兴趣技能 | 60% | KP 可调整 |
| 信用评级 | 无硬上限 | 按职业设定合理范围 |

## 五、SAN 值机制

### 初始 SAN = POW 值

### SAN 损失

| 情景 | 损失范围 |
|------|----------|
| 见到尸体 | 0/1d2 |
| 见到超自然实体（弱） | 0/1d4 |
| 见到超自然实体（中等） | 0/1d6 |
| 见到神话生物 | 1/1d10 |
| 见到旧日支配者 | 1d10/1d100 |

### 疯狂阈值

- **短期疯狂**：单次损失 ≥5 SAN
- **长期疯狂**：单日损失 ≥1/5 当前 SAN
- **不定期疯狂**：SAN 降至 0
